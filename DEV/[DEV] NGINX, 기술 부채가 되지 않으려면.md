### mirror

![](../images/[DEV]%20NGINX,%20기술%20부채가%20되지%20않으려면_16.png)

nginx 설정은 업데이트를 한다. 기존 설정이 변경되다보니 항상 걱정하게 된다.

mirror 는 들어온 요청을 그대로 복제해서 또 다른 upstream 서버 / block 으로 보내주는 역할을 한다.

이를 통해 기존 로직을 손대지 않고 설정 테스트가 가능하다.
